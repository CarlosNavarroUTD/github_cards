Muestra múltiples tarjetas de GitHub en un carrusel usando Swiper.js

== Description ==

GitHub Cards Carousel es un plugin de WordPress que te permite mostrar información de repositorios de GitHub en un elegante carrusel. Utiliza la API de GitHub para obtener datos actualizados de los repositorios y Swiper.js para crear un carrusel interactivo.

== Installation ==

1. Sube la carpeta `github-cards-carousel` al directorio `/wp-content/plugins/`
2. Activa el plugin a través del menú 'Plugins' en WordPress
3. Ve a Configuración -> GitHub Cards y añade tu API Key de GitHub
4. Usa el shortcode [github_cards repos="usuario/repo1, usuario/repo2"] en tus posts o páginas

== Frequently Asked Questions ==

= ¿Cómo obtengo una API Key de GitHub? =

Puedes crear una API Key en tu cuenta de GitHub yendo a Settings -> Developer settings -> Personal access tokens.
